ProMaIDes GIS Tools (version {version})

is a set of tools to make working with the software ProMaIDes easier. Therefore, it provides a collection of tools to import GIS data into ProMaIDes and vice versa. Model set-up and result visualisation gets much easier.

ProMaIDes

is a decision support system for flood risk analysis and the evaluation of flood risk mitigation measures in riverine and coastal area. It is developed by Daniel Bachmann (University of applied sciences Mmagdeburg-Stendal) and the Institute of hydraulic engineering at RWTH Aachen University. The freeware software package provides a comprehensive set of tools

  • 1D/2D hydraulic modeling
  • Analysisi of consequences for economic damages, affected and endangered persons, ecological consequences and critical infrastructure
  • Monte-Carlo based dike and dune fragility analysis
  • Integrating flood risk analysis
  • MADM analysis
